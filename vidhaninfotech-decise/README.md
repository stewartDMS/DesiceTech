### How do I get set up? ###

* Summary of set up
    Node Version  :  16.15.1
    - For Install node packeage using below command
        npm run build
    
    - For Run Nodejs Project using below command
        npm run serve

    - Clear cache then run this command
        npm cache clean --force

    - Dependencies error then solve this error follow this steps
        1. Remove node_modules and package-lock.json file 
        2. After removing this both file then run this code 
            npm cache clean --f
        3. After Install node packages using this code 
            npm run build
        4. After installing this packages
            npm run serve
        5. If any issue occur then please check node version 
            - we are using node version 16.15.1
